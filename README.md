翻訳者は随時募集中です。翻訳したドキュメントがあって、このリポジトリに追加したい場合は、Pull Requestを送って下さい。作業のコンフリクトを避けたい場合は、プロジェクトメンバーに事前に連絡していただければ、既知のコンフリクトがあれば、お知らせします。

既存の翻訳文書に対する修正案もPRでお待ちしています。
